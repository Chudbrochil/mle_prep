# Large Model Training

A multi-layer neural network training pipeline with train/validation loops.

## Expected Output
When working correctly:
1. Train for 5 epochs
2. Print decreasing train loss each epoch
3. Report validation loss and accuracy
4. Exit cleanly with "Training complete."

## Files
- `train_large_model.py` — Training pipeline script

## Usage
```bash
python train_large_model.py
```
